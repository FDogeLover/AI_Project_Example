# 2025-05-15 加入事前回顾和事后优化机制

## 做了什么
在 AGENTS.md 中新增两个机制：

### 事前自检（Self Review）
动手之前先查：
1. lab_book/mistakes/ → 别犯老错
2. lab_book/patterns/ → 用现成模式
3. 项目 notes/ → 了解上下文
4. CONVENTIONS.md → 遵守约定

### 事后自动优化
做完之后检查工作台本身：
1. 错误记录更新了吗？
2. 模式发现了吗？过时的修正了吗？
3. CONVENTIONS 需要更新吗？
4. 有值得做成 skill 的吗？
5. 工具箱需要更新吗？

## 为什么
之前的系统只记录不回顾——记录是事后写日记，但没有事前翻日记。
现在变成：先翻旧日记 → 干活 → 写新日记 → 检查要不要改旧日记

## 效果
形成完整闭环：
回顾 → 行动 → 记录 → 优化 → 下次回顾时能读到更准确的记录

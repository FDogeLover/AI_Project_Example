# 2025-05-15 skill_factory 项目接入

## 项目基本信息
- 名称: skill_factory
- 类型: mixed（知识库 + 流程规范）
- 阶段: 刚开始
- 技术栈: 纯 Markdown，不依赖任何编程语言

## 用户提供的信息
- Skill = AI Agent 的可复用知识模块
- 目标 = 教 AI 如何在任何工程中为自己生成 skill
- 两种机制：
  - A: AI 主动观察用户习惯，发现可复用模式，自动生成
  - B: 用户提出需求，AI 详细构建
- 不是独立工具，是示范手册

## 初始设计决策
- 用 Markdown 手册而非程序——AI 读了就能用
- 5 个手册章节：何时创建 → 如何观察 → 如何构建 → 标准结构 → 质量检查
- 两个示例 skill 分别展示机制 A 和机制 B
- 一个通用模板供快速创建

## 后续关注点
- 实际使用中手册是否需要调整
- 是否需要更多示例
- 是否需要跨工程的 skill 迁移机制

# 2025-05-15 新增四项机制 + 导出功能

## 新增的四个脚本

### 1. status_board.py（状态看板）
集中显示：项目状态、inbox 积压、最近活动、待处理事项。
一眼看清工作台全貌。

### 2. knowledge_guard.py（知识保鲜）
检查：过时的记录、pattern 之间的矛盾、未解决的 mistakes。
定期运行，保持知识库健康。

### 3. session_resume.py（会话恢复）
生成"上次工作快照"：活跃项目、最新记录、待处理事项。
新 session 进来快速接续。

### 4. export_framework.py（框架导出）
导出框架本身（AGENTS.md + lab_book 空结构 + tools/），不含具体项目。
供分享给他人，对方拿到后可以个性化使用。

## 导出内容
```
✅ AGENTS.md、CONVENTIONS.md、.gitignore
✅ lab_book/（空结构 + 索引）
✅ tools/（完整）
✅ workspace/README.md
✅ QUICKSTART.md（快速开始指南）

❌ 不含具体项目、数据、实验记录
```

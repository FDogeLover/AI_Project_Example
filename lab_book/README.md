# 实验记录本

记录**全局认知积累**——跨项目通用的思考、决策、教训和模式。

## 与项目 notes/ 的区别

```
lab_book/（全局）          workspace/<项目>/notes/（项目级）
─────────────────────     ──────────────────────────────
所有项目共享的知识          只跟这个项目有关的知识
通用的模式和教训            特定于这个项目的技术决策
AI 每次进来看这里            AI 看具体项目时才看这里
```

## 什么写在这里

- 跨项目通用的开发模式
- 所有项目都该避免的错误
- 通用的技术选型经验
- AI 协作方法论的改进

## 什么不写在这里

- 某个项目特有的技术细节
- 某个项目特有的进度记录
- 一次性的工作备忘

## 目录结构

```
lab_book/
├── README.md         ← 你正在读的这个文件
├── entries/          ← 按时间的通用记录条目
├── patterns/         ← 发现的通用规律和模式
└── mistakes/         ← 犯过的通用错误和教训
```

## 知识升级机制

当一个项目级经验被多次验证后，提升为全局知识：

```
项目 notes/ 中发现一个好模式
    ↓
第二个项目也出现了同样的模式
    ↓
提升到 lab_book/patterns/
    ↓
所有项目共享这个知识
```

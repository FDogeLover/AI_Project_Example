# 项目接入日志模板

当完成一个新项目的接入流程后，用此模板记录。

```markdown
# <日期> <项目名> 接入

## 项目基本信息
- 名称:
- 类型: （backend / frontend / ml / embedded / ...）
- 阶段:
- 技术栈:

## 接入过程
<!-- 你是怎么了解这个项目的？读了哪些文件？ -->

## 用户提供的信息
<!-- 用户回答了哪些关键问题？ -->

## 你的初始建议
<!-- 你提出了哪些建议？用户的反馈？ -->

## 后续关注点
<!-- 你标记了哪些需要持续关注的事项？ -->

## 关联记录
<!-- 链接到相关的 lab_book/ 条目 -->
```

# 新项目接入模板

当 workspace/ 下出现新项目时，复制此模板并填写。

```markdown
# <项目名>

> type: backend | frontend | ml | embedded | script | note | mixed
> status: active
> created: YYYY-MM-DD
> last_active: YYYY-MM-DD

## 做什么
<!-- 一句话说明这个项目的目标 -->

## 技术栈
<!-- 列出使用的核心技术 -->

## 项目背景
<!-- 为什么要做这个项目？解决什么问题？ -->

## 当前阶段
<!-- 刚开始 / 开发中 / 测试中 / 已上线 -->

## 关键约束
<!-- 技术约束、时间约束、硬件约束等 -->

## 已知风险
<!-- 可能出问题的地方 -->

## 关键决策
<!-- 链接到 lab_book/ 中的决策记录 -->

## 进度日志
<!-- 按时间记录关键进展 -->
```
